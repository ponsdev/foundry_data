Important Changes

Below are just some of the changes to the module that should be called out. The full changelog is also available here.

v2.7.3

  • Fixed tile deletion issue in different scenes

v2.7.2

  • Fixed marker in wrong location after reloading
  • Fixed marker not pausing with game
  • Marker visibilities are now tied to vision

v2.7.1

  • Fixing ghosting issues with some start markers
  • Fixing turn marker showing when out of combat

v2.7.0

  • Test compatibility with Foundry v0.7.9
  • You can now select who to announce turns for: all, players, npcs, or all w/ npc names hidden
  • Options have been added to switch between announcing turns by Token name, or Actor name (for those subterfuge moments)
  • Fixed a console error when moving a token outside of combat

v2.6.9

  • Fixed a console error when moving a token outside of combat

v2.6.8

  • Test compatibility with Foundry v0.6.2
  • Start marker if enabled will no longer show up until after a token has moved for the first time on it's turn
  • Video files (webm, ogg, & mp4) should now properly display in the marker previewer in settings
